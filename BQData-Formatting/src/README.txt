Baseline Question Data Formatting
--------------------------------------------------------------------------------------
unix> python BQDataFormat.py 
--------------------------------------------------------------------------------------
一、目的
   该脚本对员工关于非选择题baseline questions的答复进行统一格式化，便于后续数据分析。

二、所需问卷
   《baseline question data》

三、所需变量
   填写答案

四、生成文件
   将格式化后的“填写答案”一栏覆盖原数据，并集中其他需要手动操作的条目至最后

五、格式化规则
   0.首先去除所有空格、标点、特殊字符
   1.数字统一为阿拉伯数字形式，如6万、六万、陆万 => 50000
   2.（将近）x（多）万以上/以下/以内/左右/多 => x0000
   3.十、十几、十几万 => 100000
   4.几十万、几百万、几千万 => 归为手动
   5.a（万）到b万 => (a+b)/2*10000
   6.小数字如17、19、26等 => 170000、190000、260000
   7.首尾有效字符前后出现其他字符的，删除字符的前后序字符。
     如“好像是20000多” => 20000;“1号30万” => 300000
   8.不知道/不清楚 => 不确定
   9.空值 => -
   10.typo或者文不对题的，归为手动
